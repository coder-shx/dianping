# dianping

本学期的课程Project是一个**渐进式迭代开发的餐厅点评系统**，采用前后端分离的Web开发模式，项目的开发过程中可能接触到以下技术栈：

• 协作开发：Git、华为云CodeArts平台

• 数据库：MySQL

• 前端技术：HTML、CSS、JavaScript、Vue 或 React 等前端框架

• 后端技术：RESTful API 设计、SpringBoot(Java) 或 FastAPI(Python) 等后端框架
